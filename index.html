VIA

VIA is an application developed by Rainbow Beard productions for use with Android devices

Download project Download website Fork me on GitHub

Description

VIA is a mobile app that utilizes your camera and broadband access to answer questions you have about the world around you.

Motivation

Orignally, we were inspired by the use of Glass. This seemed to be an app that lent itself directly to Glass and the features inherent in that technology. However, after Glass proved to be unsupported and overly complicated for this project, we decided to pivot. We reviewed our concept as a mobile device and realized that it was still entirely sound and feasible.

The app concept was initially targeted at art. We then considered utilizing GPS features to add geographical data. But truly, any and all this data could be combined to answer and inform on almost everything around you.
